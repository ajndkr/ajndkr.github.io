Last year, I built my blog website using Jekyll but the whole experience wasn't 
super enjoyable for me. Plus, I couldn't find a lot of motivation to write any
posts.

This time, I decided to go with Hugo and so far, I'm thoroughly enjoying the
experience! I will create a new post to discuss it further.

## What's new?

So in my second attempt, I want to be more active in writing blogs as well as 
share some of my personal notes on topics I find interesting. I will be using 
[Dendron](https://www.dendron.so/) on VSCode to build my notes. I have still figure 
out a way to export Dendron notes to Hugo. It doesn't appear to be trivial, so 
I'll most likely write a post on it once I'm done.
