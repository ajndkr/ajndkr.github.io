# ajndkr.com

A personal blog with a terminal-inspired aesthetic. Looks like a TUI, works like a web app.

## Stack

- React 18 + TypeScript
- Vite
- Tailwind CSS + shadcn/ui
- TanStack Router
- MDX

## Development

```bash
pnpm install
pnpm dev
```

## Build

```bash
pnpm build
pnpm preview
```

## Structure

```
src/
├── routes/       # File-based routing
├── components/   # UI components
└── content/      # MDX blog posts
```
